## Fruit Sharing
My friend Chitoti picked $C$ chirimoris from the trees in her garden. There are $N$ persons in the family and they want to share the fruits such that everyone gets equal number of chirimoris. They cannot cut the fruits.

How many chirimoris will each person get?

How many chirimoris will remain?

### Input
The first line of the input contains $C$ - the number of chirimoris to share.
The second line contains $N$ - the number of family members.


### Output
In the first line of the output, print the number of fruits that each person gets.

In the second line, print the number of remaining fruits.

### Constraints
* $1 \le C, N \le 100$

### Example input
    23
    5

### Example output
    4
    3

### Explanation of the example
$23 = 5 \cdot 4 + 3$, so each person gets $4$ fruits and $3$ of them remain.
