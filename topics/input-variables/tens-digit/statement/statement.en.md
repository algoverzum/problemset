## Tens Digit
You are given a positive integer number. Print its tens digit, or, in other words, the second digit from the back. If the number has only one digit, you should print $0$.

### Input
The first line of the input contains a single integer number $N$.


### Output
Print a single number, the tens digit of $N$.

### Constraints
* $1 \le N \le 100\,000$

### Example input
    1234

### Example output
    3

### Explanation of the example
In the number $1234$, the tens digit is the digit before the last, which is $3$.
