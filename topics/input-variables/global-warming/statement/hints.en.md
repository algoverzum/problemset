### Hint 1
We can create a variable named `t` that can store an integer like this:

    int t;

We can read a value into this variable like this:

    cin >> t;

Reminder: we can print something with the `cout <<` statement. We can use variables in mathematical operations just like we use concrete numbers.


### Hint 2
For example, if we want to get a number that is one greater than `t`, we can write `t + 1`.

### Hint 3
    cout << t - 1 << " " << t + 1;
