## Global warming
Welcome to our planet! We have observed that the average temperature here increases by exactly 1 degree every year. For example, if this year's average temperature is 10 degrees, last year's was 9, and next year's will be 11. Can you write a program that *reads* an integer, which represents the average temperature of a year, and prints out the average temperatures of the previous and next years?

### Input
The input consists of a single integer $T$, the average temperature of the given year.

### Output
You should output two numbers separated by a space, the average temperature of the previous year and the average temperature of the next year.

### Constraints
* $1 \le T \le 100$

### Example input
    34

### Example output
    33 35

### Second example input
    1

### Second example output
    0 2
