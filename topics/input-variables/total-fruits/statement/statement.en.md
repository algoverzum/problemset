## Total Fruits
There are many exotic fruits on our planet. One fine day, my three friends, Akiko, Bogiga, and Chitoti, went fruit picking. Akiko picked $A$ angots, Bogiga picked $B$ bererees, and Chitoti picked $C$ chirimoris. Now they are curious to know how many fruits they picked altogether. Can you write a program that answers this question given the number of each type of fruit?

### Input
The input contains three integers: $A$, $B$, and $C$ - the number of angots, bererees, and chirimoris picked, respectively.

### Output
Print a single number, the total number of fruits picked by the three friends.

### Constraints
* $1 \le A, B, C \le 100$

### Example input
    13 5 24

### Example output
    42
