## Duration
You are given a duration in seconds. Print it in the following format:

    D H M S

where $D$ means days, $H$ means hours, $M$ means minutes and $S$ means seconds.

### Input
The first line of the input contains $N$ - the number of seconds.


### Output
Print four numbers in one line, the duration in days, hours, minutes, seconds format.

### Constraints
* $1 \le N \le 1\,000\,000$

### Example input
    93665

### Example output
    1 2 1 5

### Explanation of the example
$93\,665$ seconds equals $1$ day and $2$ hours and $1$ minute and $5$ seconds.
