## Sum A B
Write a function that calculates the sum of the numbers $A$ to $B$. $A$ and $B$ are integers, and $A \le B$.

### Constraints
* $-10\,000 \le A \le B \le 10\,000$

### Template
Start from the predefined template code. Do not change anything in the main program, otherwise it will not be accepted. You need to define a function called `sum_between`, which takes two parameters: $A$ and $B$. The return value must be the sum of the numbers from $A$ to $B$, for example `sum_between(4,6) = 4 + 5 + 6 = 15`.
