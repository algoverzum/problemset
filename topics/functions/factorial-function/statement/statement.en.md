## Factorial Function
Write a function that computes $N$ factorial, i.e. the product of the numbers from 1 to $N$ ($N! = 1 \cdot 2 \cdot \ldots \cdot N$.)

### Constraints
* $1 \le N \le 15$

### Template
Start from the predefined template code. Do not change anything in the main program, otherwise it will not be accepted. You have to define a function called `factorial` which takes $N$ as a parameter. The return value should be the product of the numbers $1$ to $N$. For example, `factorial(4) = 1 * 2 * 3 * 4 = 24`.
