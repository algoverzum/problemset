## Összeg A-tól B-ig
Írj függvényt, ami kiszámolja $N$ faktoriálist, azaz 1-től $N$-ig a számok szorzatát!

### Korlátok
* $0 \le N \le 15$

### Sablon
Indulj ki az alábbi sablonból (másold be ezt a szerkesztőbe)! Ne változtass a `main` függvényben semmit, mert különben nem lesz elfogadva. A `factorial` függvény belsejét kell megírnod, ami paraméterként kapja az $N$ értékét.

```cpp
#include <iostream>
using namespace std;

long long factorial(int n) {

}

int main() {
    for (int i = 1; i <= 15; i++) {
        cout << factorial(i) << "\n";
    }
}
```
