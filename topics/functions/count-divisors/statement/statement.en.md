## Count Divisors
We are exploring the temple of a lost alien civilization. To interpret the mysterious texts on the walls, it is essential to know the number of divisors that the numbers have at the beginning of each line. Write a function to help the researchers to find the number of divisors for a given number.

### Template
Start from the pre-made template code! Do not change anything in the main program, otherwise it will not be accepted. You need to write the function `count-divisors`, which takes as a parameter the number whose divisors we are looking for, and returns the number of divisors of that number. (From the purpose of the exercise, we count as divisors both 1 and the number itself.)

### Constraints
* The number obtained is always between $1$ and $100000$.

### Example
25 has 3 divisors: 1, 5, 25.
