## Intersection
The Enterprise and the Voyager are searching for new intelligent species in two different sectors. Starfleet scientists want to know which species have been detected by both ships!

Write a function for a data-processing computer that will tell you which species have been detected by both Enterprise and Voyager!

### Template
Start from the template code you've prepared. Don't change anything in the main program or it won't be accepted. You need to write the inside of the `intesection` function, which will take as parameters the array of numbers in ascending order, and return the array of numbers in both in ascending order.

### Constraints
* $1 \le N, M \le 100$, where $N$ and $M$ are the sizes of the two arrays.
* All numbers in the arrays are between $0$ and $10^6$.

### Example
If one of the arrays is $4, 6, 8, 9$ and the other is $3, 5, 6, 7, 9, 11$ then their intersection is $6, 9$.
