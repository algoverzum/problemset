## Odd Numbers
The Federation was analysing a data stream from an unknown planet. According to Starfleet scientists, only certain numbers -- odd values -- carry information.
Write a function that takes a sequence of integers and returns a new sequence containing only the odd numbers so that the scientists can analyze them further!

### Template
Start from the pre-made template code! Do not change anything in the main program, otherwise it will not be accepted. You need to write the inside of the `odd_numbers` function, which takes the sequence of numbers as a parameter and returns the sequence of odd numbers.

### Constraints
* $1 \le N \le 1000$, where $N$ is the length of the stream.
* All numbers in the data stream are between $0$ and $1\,000\,000$.

### Example
Only the odd numbers of $0, 7, 12, 36, 19, 1, 24$ should be returned in the original order, i.e. $7, 19, 1$.
