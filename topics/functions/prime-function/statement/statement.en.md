## Prime Function
Write a function that decides whether a positive number is prime or not. (Primes are integers greater than 1 that have no divisors other than 1 and themselves.)

### Constraints
* $1 \le N \le 1000$

### Template
Start from the predefined template code. Do not change anything in the main program, otherwise it will not be accepted. You have to define a function called `prime` which takes $N$ as a parameter. The return value should be "true" if $N$ is prime, otherwise it should be "false".
