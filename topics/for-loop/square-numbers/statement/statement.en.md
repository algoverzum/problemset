## Square Numbers
You are given a positive integer $N$. Can you print the first $N$ positive square numbers?

### Input
The input contains a single integer number $N$.

### Output
Print $N$ numbers in separate lines, the first $N$ square numbers.

### Constraints
* $1 \le N \le 100$

### Example input
    4

### Example output
    1
    4
    9
    16

### Explanation of the example
The square numbers are $1 \cdot 1 = 1$, $2 \cdot 2 = 4$, $3 \cdot 3=9$, ...  $N=4$ in the example, so we need to print them until $4 \cdot 4 = 16$.
