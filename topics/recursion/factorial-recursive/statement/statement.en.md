## Factorial Recursively
Write a **recursive** function to calculate the factorial of $N$, which is the product of numbers from 1 to $N$!

### Constraints
* $0 \le N \le 15$

### Template
Start with the provided template code! Do not modify the main program; otherwise, your solution will not be accepted. You need to implement the body of the `factorial` function, which takes $N$ as a parameter.
