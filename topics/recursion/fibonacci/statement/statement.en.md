## Fibonacci recursively
The USS Enterprise has detected a mysterious signal from a distant nebula. To decode the signal, the Fibonacci sequence was used. 
Write a **recursive** function that computes the $N$-th Fibonacci number.
The first two elements of the Fibonacci sequence are 1, and then each element is the sum of the previous two: 1, 1, 2, 3, 5, 8, 13, 21, ...

### Constraints
* $1 \le N \le 30$

### Template
Start with the provided code template! Do not modify the main program; otherwise, your solution will not be accepted. You need to implement the body of the `fibonacci` function, which takes $N$ as a parameter.
