## Digit Sum
Write an *elegant* function to calculate the sum of the digits of a positive integer $N$!

### Constraints
* $1 \le N \le 1\,000\,000\,000$

### Template
Start with the provided template code! Do not modify the main program; otherwise, your solution will not be accepted. You need to implement the body of the `digit_sum` function, which takes $N$ as a parameter.
