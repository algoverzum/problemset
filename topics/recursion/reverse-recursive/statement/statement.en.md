## Reverse Recursively
The Jedi Council has prepared a special test for the young Padawans to prove their ability to master the Force. In the hidden archives of the Jedi Temple, Master Yoda challenges you to reverse a series of numbers. The difficulty? You can't use arrays (such as lists or vectors), you can only rely on the pure flow of the Force - aka **recursion**.

### Input
Each row of the input is an integer: $V_i$.
The last row has 0, and all the numbers before it are positive.

### Output
Write the same number of lines as the input, one number in each line, with the numbers in the input in reverse order (backwards).
Do not use an arrays (vectors, lists, etc.)

### Constraints
* $1 \le N \le 100$, where $N$ is the number of rows of input.
* $1 \le V_i \le 10^9$, where $V_i$ is the number in the $i$th row of the input.

### Example input
    1
    11
    7
    0

### Example output
    0
    7
    11
    1

### Explanation of the example
You have to write the numbers backwards... Do not use an array (vector, list, etc.)
