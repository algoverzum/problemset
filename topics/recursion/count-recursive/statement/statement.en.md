## Count Recursive
The USS Enterprise has received a strange text signal from a distant planet. The signal contained a single word, which may have been the key to the alien civilization's communications. However, the word was encoded by repetitions of a specific letter, and the crew had to count the number of times the letter occurred in the word to reveal the hidden message.
Can you solve this task **recursively**?

### Template
Start from the pre-made template code! Do not change anything in the main program, otherwise it will not be accepted. You need to write the inside of the `count` function, which takes the string $S$ and the letter $c$ as parameters, and returns how many times is the letter present in the string.

### Constraints
The $S$ string is up to 20 characters long and contains lower case only.

### Example
For example, in the word "alaska" the letter 'a' appears 3 times.
