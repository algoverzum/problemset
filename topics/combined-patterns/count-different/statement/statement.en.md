## Count Different
The Galactic Research Union took samples of soil on a distant planet. Each sample is marked with a number based on what it is made of. How many different materials did we find in our sampling?

### Input
The first line of the input contains a single integer: $N$.
The second line of the input contains $N$ integers: $A_1, A_2, \ldots, A_N$, the numbers marking the materials of the samples.

### Output
Print a single integer, the number of different materials among the samples of soil.

### Constraints
* $1 \le N \le 1000$
* $1 \le A_i \le 1000$

### Example input
    5
    1 2 3 2 1

### Example output
    3

### Explanation of the example
There are 3 different elements in the input: 1, 2, 3.
