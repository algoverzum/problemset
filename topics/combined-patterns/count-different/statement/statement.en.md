## Count Different
The Galactic Research Union took samples of soil on a distant planet. Each sample is marked with a number based on what it is made of. How many different materials did we find in our sampling.

### Input
The first line of the input contains a single integer: $N$ 
The second line of the input contains $N$ integers $A$ .

### Output
Print a single number to see how many different elements there are in the number line.

### Constraints
* $1 \le N \le 10000$
* $1 \le A \le 10000$

### Example input
    5
    1 2 3 2 1

### Example output
    3

### Explanation of the example
3 different elements: 1, 2, 3
