## Treasure Planet
The Empire has hidden a secret treasure on the edge of the galaxy, but its exact location is encoded with the names of planets. Each planet holds a clue, but only those with the letter "o" in their name will guide you to the treasure.
Your mission: determine how many planets on the list contain the letter "o". May the Force be with you as you uncover the treasure's location!

### Input
The first line of the input contains $N$ - the number of words.
The second line contains $N$ space separated words. Each word contains only lowercase letters.

### Output
Print a single number, the number of words containing "o".

### Constraints
* $1 \le N \le 100$
* Each word contains only lowercase letters.

### Example input
    7
    sun weekend oreo borg hell babie soon

### Example output
    3

### Explanation of the example
Three words containd "o": oero, borg, soon.
