## Decimal to Binary
Write a function that converts a positive decimal (base-10) integer into its binary (base-2) representation.

### Template
Start from the provided template code. Do **not** modify the main program, otherwise the solution will not be accepted. You need to implement the `dec2bin` function, which receives the integer as a parameter and returns the result as a string.

### Constraints
* $1 \le N \le 10^9$

### Example
The decimal number 238 is `11101110` in binary.
