## Anagram
Write a function that determines whether two strings $S_1$ and $S_2$ are anagrams of each other. Two strings are anagrams if they consist of the same letters with the same frequencies, but possibly in a different order.

The function must return a boolean (true/false) value.
The strings contain only lowercase letters of the English alphabet.

### Template
Start from the provided template code. Do not modify the main program, otherwise the solution will not be accepted. You only need to implement the anagramma function, which receives the two strings as parameters and returns a boolean value.

### Constraints
* $1 \le |S_1|, |S_2| \le 100$
* $S_1$ and $S_2$ contain only lowercase letters of the English alphabet.

### Example 1
"satu" and "tusa" are anagrams.

### Example 2
"satuk" and "suta" are not anagrams.

### Example 3
"satu" and "tusi" are not anagrams.
