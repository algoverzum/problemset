## World Cup
A sport event is held in June of every year whose remainder when divided by 4 is 2. Suppose that it is now January of the year $Y$. In what year will this sport event be held next time?

### Input
The first line of the input contains $Y$ - the current year.

### Output
Print the answer.

### Constraints
* $2000 \le Y \le 3000$

### Example input 1
    2022

### Example output 1
    2022

### Explanation of the example 1
The remainder when 2022 is divided by 4 is 2, so if it is now January of 2022, the next games will be held in June of the same year.

### Example input 2
    2023

### Example output 2
    2026

### Example input 3
    3000

### Example output 3
    3002
