## BC sorrend
In the Outer Rim of the galaxy, a mysterious artifact was discovered: The Codex of Sabers. It was said to contain the wisdom of ancient Jedi Masters, but its secrets were locked in a jumbled string of Aurebesh characters. Whoever could sort the letters into the correct order would unlock its hidden power - an ancient lightsaber crystal of immense strength. Let's try the alphabetic order to get at least something!
Even in a galaxy torn by conflict, sometimes the greatest power lies in putting things... in order.

### Input
The only line of the input contains $S$ - the jumbeld string. It contains only upper case English letters. 

### Output
Print a single string, containing the letters of $S$ in alphabetic order.

### Constraints
* $1 \le $ the length of $S$ $ \le 1000$
* $S$ contains only upper case English letters

### Example input
    DLFJWBQKZB
    
### Example output
    BBDFJKLQWZ

### Explanation of the example
Just the alphabetic order.
