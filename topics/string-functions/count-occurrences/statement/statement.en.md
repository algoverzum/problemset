## Count Occurrences
In the language of an alien species on an alien planet, the "beauty" of words plays an important cultural role. Humanity's diplomats have discovered that the beauty of a word in this language can be expressed by an integer and is exactly equal to the number of times the character string "obo" appears in the word. Help the diplomats to determine the beauty of certain words.

### Input
The first line of the input contains $S$ - the given word to be examined.

### Output
Print a single integer, the beauty of the given word.

### Constraints
* $3 \le $ length of $S \le 256$
* S contains only lower case letters of the English alphabet

### Example input
    golobotorobo

### Example output
    2

### Explanation of the example
In the given word, the substring "obo" appears twice: gol**obo**tor**obo**
