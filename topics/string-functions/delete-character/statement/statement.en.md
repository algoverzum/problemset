## Delete Character
As part of the "NOVA" space program, we received an encrypted message from a distant planet. The message contains a word that serves as a special code, but it is obscured by noise. To prepare the word for decoding, all occurrences of a specific letter, which may appear multiple times, must be removed from the message. Can you write a program for this task?

### Input
The first line of the input is a word $S$.
The second line contains a $C$ character.

### Output
Print the $S$ string, after removing every occurrence of the $C$ character. It may become an empty string and then you should print an empty line.

### Limits
* The word $S$ consists of lowercase letters of the English alphabet and is up to 1000 characters long.
* The character $C$ is a lowercase letter of the English alphabet.

### Example input
    delete
    e

### Example output
    dlt
