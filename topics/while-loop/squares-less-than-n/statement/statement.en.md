## Squares Less Than N
Can you list all the positive square numbers that are smaller than a given number integer $N$? (A square number is an integer number multiplied with itself, e.g. $1 \cdot 1 = 1, 2 \cdot 2 = 4$, etc.)

### Input
The input contains a single integer $N$.

### Output
You should print all the square numbers that are strictly smaller than $N$ in increasing order, one number in each line.

### Constraints
* $1 \le N \le 10\,000$

### Example input
    25

### Example output
    1
    4
    9
    16

### Explanation of the example
The first four square numbers are 1, 4, 9, 16. The next one, 25 is not smaller than 25, so we should not print it.
