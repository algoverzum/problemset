## Stop At Zero
Mo Seis Lee's new larger spaceport is complete. We know how many spaceships are coming in the next few days. On the first day when none arrives (i.e. zero arrivals), we can go on vacation. How many days do we have to work until then?

### Input
The input contains integers, one number per line. The number in row $i$ is the number of spacecraft arriving on day $i$. The last line is guaranteed to be zero.

### Output
Print a single number, the number of working days before the first day off.

### Constraints
* the $i$-th line contains an integer $S_i$, such that $0 \le S_i \le 100$.
* the input contains $N$ lines, where $1 \le N \le 100$.

### Example input
    5
    10
    7
    1
    3
    8
    0

### Example output
    6

### Explanation of the example
The first day off comes after 6 working days.
