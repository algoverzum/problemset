## Digits Sum
Given a positive integer, output the sum of its digits.
The input number has at most 9 digits.

### Input
The input consists of a single integer $N$.

### Output
Print a single integer - the sum of the digits of the given number.

### Constraints
* $1 \le N < 10^9$

### Example input 1
    12345678

### Example output 1
    36

### Explanation of the example
The sum of the digits is $1+2+3+4+5+6+7+8=36$.

### Example input 2
    10

### Example output 2
    1
