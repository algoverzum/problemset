## Names List
The crew list on your spaceship was not kept up to date, the names are in the order they came on board and the system is becoming unintuitive to you. Arrange the list in alphabetical ascending order.

### Input
The first line of the input is a single integer: $N$, the crew size.
The second line of the input contains $N$ strings separated by spaces: $S_1, S_2, \ldots S_N$, the names of the crew.

### Output
Print $N$ strings, the names of the crew in alphabetical order.

### Constraints
* $1 \le N \le 1000$
* The names contain lowercase letters of the english alphabet, and are at most 1000 characters long.

### Example input
    6
    luke leia han chewbacca kenobi vader

### Example output
    chewbacca han kenobi leia luke vader 

### Explanation of the example
c < h < k < le < lu < v
