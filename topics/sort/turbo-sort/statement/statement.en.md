## Turbo Sort
You are given a list of numbers, which must be sorted in non-decreasing order.

### Input
The first line contains an integer $N$ - the number of elements in the list.
Then follow $N$ lines.
Each of the next lines contains a single integer $A_i$, an element of the list.

### Output
Print the given numbers in non-decreasing order.
Each number should be printed on a separate line.

### Constraints
* $1 \le N \le 10^6$
* $0 \le A_i \le 10^6$ for every $i$

### Example input
    5
    5
    3
    6
    7
    1

### Example output
    1
    3
    5
    6
    7
