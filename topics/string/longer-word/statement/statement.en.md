## Longer word
I found two envelopes in the letterbox, each with a word written on it. I wonder which is longer?

### Input
The input consists of two lines, each with one word. 

### Output
You have to print one word, the longer one. If the two words are the same length, write the first one.

### Constraints
* both words contains at most 20 characters and at least 1

### Example 1 Input
    petulant
    concubine

### Example output 1
    concubine

### Explanation of example 1
"petulant" is 8 characters long, "concubine" is 9 characters long, so "concubine" is the longer one.

### Example 2 Input
    petulant
    volition

### Example output 2
    petulant

### Explanation of example 2
Both words consist of 8 characters, so the first one should be printed.

### Example 3 Input
    megalopolis
    desiccated

### Example output 3
    megalopolis

### Explanation of example 3
"megalopolis" is 11 characters long, "desiccated" is 10 characters long, so "megalopolis" is the longer one.
