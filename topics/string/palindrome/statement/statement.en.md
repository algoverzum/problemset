## Palindrom
Extraterrestrial diplomats compare different languages and need help deciding which word is a palindrome. A word is a palindrome if reading it backwards gives the same word. Help them! For a given word, print "YES" or "NO" depending on whether it is a palindrome.

### Input
The first line of the input is a single string: $S$.

### Output
In the first and only line of the output, you must print "YES" (all capital letters) if the word is a palindrome and "NO" if it is not.

### Constraints
* $1 \le |S| \le 256$

### Example input
    ufotofu

### Example output
    YES

### Explanation of the example
The word ufotofu read backwards is also ufotofu.
