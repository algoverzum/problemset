## Palindrom
Extraterrestrial diplomats compare different languages and need help deciding which word is a palindrome. Help them! For a given word, decide whether reading it backwards gives the same word and print "YES" or "NO" depending on whether you get the same word.

### Input
The first line of the input is a single string: $S$

### Output
In the first and only line of the output, you must print "YES" if the word is a palindrome and "NO" if it is not

### Constraints
* $1 \le |S| \le 256$

### Example input
    ufotofu

### Example output
    YES

### Explanation of the example
The word ufotofu is also read backwards as ufotofu.
