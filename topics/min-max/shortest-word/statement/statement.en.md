## Shortest Word
You want to enter the treasury of an ancient civilisation. During your adventures you have already obtained the list containing the keyword, but there are several words on the list as misleading clues. Fortunately, you know that the keyword is the shortest word on the list. Find the keyword!

### Input
The first line of the input contains a single integer: $N$ 
The second line contains $N$ words separated by a space.

### Output
Print the shortest word in the input. If there is more than one shortest word, Print the one that occurred first.

### Constraints
* $1 \le N \le 100$

### Example input
    6
    alma kecske uborka gesztenye kutya kert

### Example output
    alma
