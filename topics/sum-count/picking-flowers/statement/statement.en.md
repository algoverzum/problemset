## Picking flowers
Aladar is picking flowers. He sees $N$ flowers in the meadow, he knows how many petals each one has (these numbers are listed in the input). He picks only the flowers with odd petals. How many flowers remain in the meadow?  
The first number tells you how many flowers you see, followed by the number of petals each flower has.

### Input
The first line of the input is the number of flowers: $N$. Then given the number $N$, each on a separate line, is the number of petals of each flower ($S_1, S_2, \ldots, S_N$).  

### Output
You must print a single number, the number of flowers remaining in the meadow. 

### Limits
* $1 \le N \le 100$
* $1 \le S_i \le 100$

### Example input
    5
    7
    4
    3
    11
    6

### Example output
    2

### Explanation of the example
The flowers with 4 and 6 petals are left in the meadow, the others are picked by Aladar.
