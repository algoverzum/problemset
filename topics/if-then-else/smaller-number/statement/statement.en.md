## Smaller Number
I will tell you two integer numbers. Can you tell me which one is the smaller?

### Input
The first line of the input contains two integer numbers: $A$ and $B$. 


### Output
Print a single number, the smaller number in the input. If the two numbers are the same, you should print that number.

### Constraints
* $1 \le A, B \le 100$

### Example input
    13 7

### Example output
    7

### Explanation of the example
7 is smaller than 13, so we should print 7.
