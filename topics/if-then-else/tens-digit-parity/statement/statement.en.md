## Tens Digit Parity
Given a natural number as input, determine whether its second-to-last digit is even or odd.
You may assume that the number has at least two digits.
If the digit is even, the program should output 1; otherwise, output 0.

### Input
The first line of the input contains a single integer number $N$.

### Output
Print a single number — the parity of the digit in the tens place of $N$.
If it is even, print 1; otherwise, print 0.

### Constraints
* $1 \le N \le 1000\,000$

### Example 1 input
    1234

### Example 1 output
    0

### Explanation of the example
In the number 1234, the tens digit is the digit before the last, which is 3, which is odd.

### Example 2 input
    12345

### Example 2 output
    1
