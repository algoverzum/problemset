## Rock Paper Scissors
The well-known game is now played by two players to decide something (for example, who will take the free kick).
Each player can show one of three gestures: rock, paper, or scissors.

Rules:

* Rock beats scissors.
* Scissors beat paper.
* Paper beats rock.

Your program should compare the two inputs and determine which player wins.

The possible outputs are:

* 1 – if the first player wins,
* 2 – if the second player wins,
* 0 – if it’s a draw.

### Input
The first line of the input contains what the first player showed.
The second line of the input contains what the second player showed.

Print a single number:

* 1 if the first player wins,
* 2 if the second player wins,

Each of the two input lines contains only one of the following strings: "ko", "papir", or "ollo".
