## Triangle
Given three integers, determine whether they can be the sides of a triangle — in other words, whether a triangle with these side lengths can exist.

If yes, print 1; otherwise, print 0.

### Input
The input contains three lines:

* The first line contains an integer $A$, one side of the triangle.
* The second line contains an integer $B$, another side of the triangle.
* The third line contains an integer $C$, the third side of the triangle.

### Output
If a triangle with sides $A$, $B$, and $C$ can exist, print 1; otherwise, print 0.

### Constraints
* $1 \le A, B, C \le 1000$

### Example input 1
    3
    4
    5

### Example output 1
    1

### Explanation of the example
This is a right-angled triangle.

### Example input 2
    2
    3
    1

### Example output 2
    0
