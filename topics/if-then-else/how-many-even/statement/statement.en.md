## How Many Even?
There are two numbers given in the input. Can you tell how many of them are even (divisible by 2)?

### Input
The input contains two integer numbers $A$ and $B$ on separate lines.


### Output
Print the number of even numbers in the input that is 0, 1, or 2.

### Constraints
* $1 \le A, B \le 100$

### Example input
    7
    13

### Example output
    0

### Explanation of the example
Neither 7 or 13 is even, so the answer is 0.
