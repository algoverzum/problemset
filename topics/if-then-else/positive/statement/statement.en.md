## Positive?
On our planet, the numbers that are bigger than 0 are called positive numbers, and they are denoted by the letter P. The numbers not bigger than 0 are called non-positive numbers, denoted by N. You are given an integer number. Print the letter `P` if it is positive and `N` otherwise.

### Input
The input contains an single integer number, $X$.


### Output
Print one capital letter to the output, `P` if $X > 0$, and `N` otherwise.

### Constraints
* $-1000 \le X \le 1000$

### Example input
    0

### Example output
    N

### Explanation of the example
0 is not a positive number, so we should print `N`.
