## Parity
A kindergarten group consists of $N$ kids. The teacher lets them go out to the playground if they can form pairs holding hands without leaving anyone out. Can they go to the playground? If this is possible then print `1`, otherwise print `-1`.

### Input
The input contains a single integer $N$ - the number of kids.

### Output
Print `1` if the kids can go to the playground and `-1` if they can't.

### Constraints
* $1 \le N \le 100$

### Example input
    15

### Example output
    -1

### Explanation of the example
15 is odd, so the kids cannot go to the playground.
