## Leap Year
We are given an integer, and we need to determine whether it represents a leap year. According to the Gregorian calendar, a year is a leap year if it is divisible by 4, except that years divisible by 100 are not leap years unless they are also divisible by 400. If the given year is a leap year, output 1; otherwise, output 0.

### Input
The input contains a single integer $N$, the year in question.

### Output
Output `1` if the year is a leap year, otherwise output `0`.

### Constraints
* $1 \le N \le 10000$

### Example Input 1
    2024

### Example Output 1
    1

### Example Input 2
    2025

### Example Output 2
    0

### Example Input 3
    2000

### Example Output 3
    1
    
### Example Input 4
    2100

### Example Output 4
    0
