### Chess Board
I lost my boarding pass. I remembered wrongly, there were characters $X$ and $Y$ in a square, like a chessboard. Can you help me print a new one?

### Input
A single line of the input contains an integer: $N$ - the length of the side of the chessboard.

### Output
$N$ lines must be printed. In each row, there should be exactly $N$ pieces of $X$ or $Y$ such that the adjacent ones are different.
There should be $X$ in the upper left corner, i.e. the first character of the first line should be $X$!

### Constraints
* $1 \le N \le 20$

### Example input
    4

### Example output
    XYXY
    YXYX
    XYXY
    YXYX

### Explanation of the example
We need to draw a 4x4 chessboard.
