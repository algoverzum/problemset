## Arrowhead
Write a program that reads an integer $N$ and prints an arrowhead-shaped pattern made of asterisks (*).
The pattern first contains an increasing number of stars from $1$ up to $N$ in each line, then a decreasing number of stars from $N-1$ down to $1$. The line containing $N$ stars should appear only once.

### Input
The input consists of a single integer $N$, the number of stars in the middle line. 

### Output
Print the star pattern (arrowhead) as shown in the examples.

### Constraints
* $1 \le N \le 40$

### Example input
    6

### Example output
    *
    **
    ***
    ****
    *****
    ******
    *****
    ****
    ***
    **
    *
