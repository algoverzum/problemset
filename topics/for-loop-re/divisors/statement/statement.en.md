## Divisors
Write a program that reads a positive integer number and prints its divisors.

### Input
The first line of the input contains $N$ - a positive integer number.

### Output
Print the divisors of $N$ in increasing order, each number in a separate line.

### Constraints
* $1 \le N \le 100$

### Example input
    42

### Example output
    1
    2
    3
    6
    7
    14
    21
    42
