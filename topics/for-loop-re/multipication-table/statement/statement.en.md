## Multipication Table
Given $N$, write out a multiplication table of $N*N$: the first row should contain the numbers 1, 2, ..., $N$, the second row should contain twice these numbers, the third row should contain three times these numbers, and so on up to the $N$ th row.

### Input
The input is a single integer: $N$ 

### Output
You must print $N$ rows with $N$ values separated by spaces in each row, so that the values are multiples of $A$ and $B$, where $A$ is the index of the row of the value and $B$ is the index of the column of the value.

### Limits
* $1 \le N \le 100$

### Example input
    5
  
### Example output
    1 2 3 4 5
    2 4 6 8 10
    3 6 9 12 15
    4 8 12 16 20
    5 10 15 20 25
